## Overview

The Alice layout is an ergonomic keyboard design that has gained popularity for its unique split and angled layout, which aims to reduce wrist strain and improve typing comfort. This DIY build features a custom PCB with the distinctive Alice curve.

## Key Features

- **Layout**: Alice ergonomic (split angled)
- **Ergonomics**: Natural hand positioning with angled halves
- **Firmware**: QMK programmable
- **Design**: Custom PCB with through-hole and SMD components
- **Case**: 3D printed or acrylic layered construction

## Ergonomic Benefits

- Reduces ulnar deviation (wrist bending)
- Promotes neutral wrist positioning
- Angled key wells match natural finger lengths
- Split design allows shoulder-width hand positioning

## Technical Specifications

- Custom PCB with integrated microcontroller support
- Hot-swappable or soldered switch options
- Programmable keymaps via QMK
- Custom 3D printed nameplate with Greek letter design
