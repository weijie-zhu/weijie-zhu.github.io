## Overview

The Corne keyboard (CRKBD - Corne Keyboard) is a popular split ergonomic keyboard design with a 3x6 columnar staggered key layout plus 3 thumb keys per side. Originally based on the Helix design, it has become one of the most widely adopted ergonomic keyboard projects in the DIY community.

## Key Features

- **Layout**: 42-key split ergonomic (3x6 columnar + 3 thumb keys)
- **Type**: Columnar staggered ergonomic
- **Firmware**: QMK and ZMK support
- **Connection**: TRRS cable or wireless (with ZMK)
- **Design**: Compact 40% form factor per half

## Ergonomic Benefits

- **Columnar layout**: Keys aligned in columns matching natural finger movement
- **Split design**: Allows shoulder-width hand positioning
- **Tenting capability**: Can be angled for neutral wrist rotation
- **Reduced key count**: Encourages efficient keymap design with layers

## Technical Specifications

- 42 keys total (21 per side)
- 3x6 columnar staggered alpha grid
- 3 thumb keys per side
- OLED display support (optional)
- Rotary encoder support (optional)
- QMK/ZMK fully programmable
