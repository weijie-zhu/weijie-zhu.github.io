## Overview

Le Chiffre (ironically "the number" in French) is a 30% ortholinear keyboard designed by tominabox1. This build features an extremely compact layout that maximizes typing efficiency while minimizing hand movement.

## Key Features

- **Layout**: 30% ortholinear (columnar stagger)
- **Key Count**: Approximately 30 keys
- **Firmware**: QMK (officially supported in qmk_firmware repository)
- **Controller**: Pro Micro compatible
- **Design Philosophy**: Minimalist ortholinear layout for touch typing efficiency

## Technical Specifications

- Ortholinear key arrangement (keys aligned in straight columns)
- QMK firmware with full programmability
- Compact form factor reduces desk footprint
- Custom keymap support for personalized workflow
