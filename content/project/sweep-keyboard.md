## Overview

The Ferris Sweep (often just called "Sweep") is a 34-key split keyboard designed by pierrechevalier83. It features a distinctive curved layout that follows natural finger arcs, providing ergonomic typing in an ultra-compact form factor. The design is powered by QMK firmware and has also been adapted for ZMK wireless builds.

## Key Features

- **Layout**: 34-key split ergonomic (17 keys per side)
- **Design**: Curved layout following natural finger arcs
- **Firmware**: QMK and ZMK support
- **Controller**: Pro Micro compatible (dual)
- **Connection**: TRRS cable or wireless

## Ergonomic Benefits

- **Curved layout**: Keys positioned along natural finger arc paths
- **Compact form**: Minimal hand travel distance
- **Split design**: Allows customized hand separation and tenting
- **Layer-based typing**: Full functionality through efficient keymap design

## Technical Specifications

- 34 keys total (17 per side)
- Curved staggered column layout
- Dual Pro Micro controllers
- TRRS interconnect (wired builds)
- QMK/ZMK fully programmable
- Optional OLED displays
